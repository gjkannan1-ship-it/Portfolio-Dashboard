About

This dashboard runs entirely in your browser. All data is stored locally using localStorage — nothing is sent to any server.

Live stock/ETF/fund prices are fetched from Yahoo Finance via a CORS proxy. Crypto prices are from CoinGecko. Exchange rates from ExchangeRate-API.

To make it available online from anywhere, deploy the index.html file to GitHub Pages. See README.md for instructions.
